# MindFlow - Structured Chatbot Product Definition

MindFlow 是一个探索**结构化思维流**的 AI 聊天机器人产品概念。旨在通过**树状可视化交互**，解决传统线性对话在复杂任务（如代码调试、创意构思）中的上下文迷失问题。

## 🚀 快速开始 (Getting Started)

### 体验原型
直接在浏览器中打开 `prototype/index.html`。
该原型演示了以下核心功能：
1.  **会话列表**：管理多个历史会话。
2.  **思维树侧边栏**：可视化的对话分支结构。
3.  **分支切换**：点击不同节点，主对话区即时切换上下文。

## 📝 核心概念 (Core Concepts)

*   **Tree over Stream**: 虽然主界面保持流式对话体验，但底层数据结构是树（Tree）。
*   **Non-destructive Editing**: 修改提问不会覆盖历史，而是产生新的分支（Fork）。
*   **Context-Aware**: 侧边栏只高亮当前激活路径，非激活路径自动置灰。
